## Project Overview

A business website for Atelier Zaina, a wedding backdrop hire company. The site allows customers to browse services, view the gallery, and submit enquiries. It also includes an admin dashboard to view submitted enquiries.

## Pages

| Page | File |
|------|------|
| Home | `index.html` |
| Gallery | `gallery.html` |
| Services & Pricing | `services.html` |
| Contact / Enquiry | `contact.html` |
| Admin Login | `login.php` |
| Admin Dashboard | `dashboard.php` |

## Requirements

- A web server with **PHP** support 
- A modern web browser 

## How to Run Remotely
- Go to https://atelierzaina.com/

## How to Run Locally

1. Install [XAMPP](https://www.apachefriends.org/) (or any PHP-compatible server)
2. Clone or download this repository into your server's web root folder (e.g. `htdocs/atelier-zaina/`)
3. Start Apache in XAMPP
4. Open your browser and go to: `http://localhost/atelier-zaina/index.html`
